# Frontend

Se utilizo las siguientes tecnologias:

- ReactJS
- Vitejs
- CSS: Tailwind
- Paquetes: react-router
- Paquetes: axios

## Instrucciones de uso

- Tener instalado:

```
nodejs --version
```

- Instalar las dependencias

```
npm install
```

- Ejecutar comando

```
npm run dev
```

- Acceder a la URL:

[http://localhost:3000](http://localhost:3000)
